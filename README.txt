EMG ENERGY FIELD CAPTURE SYSTEM - DEMO PACKAGE
================================================

WHAT'S IN THIS FOLDER:
- index.html ........... Landing page with links to both interfaces
- field/ ............... Field operator mobile interface (PWA)
- manager/ ............. Manager triage dashboard (desktop)

HOW TO UPLOAD TO HOSTINGER:
1. Login to your Hostinger control panel
2. Go to File Manager
3. Navigate to public_html/
4. Upload this entire folder (or just the contents)
5. Access at: http://your-domain.com/index.html

YOUR SHAREABLE LINKS:
- Landing page: http://your-domain.com/
- Field operator: http://your-domain.com/field/
- Manager: http://your-domain.com/manager/

DEMO MODE NOTES:
- This is frontend-only (no backend/database)
- Data capture works but doesn't save to server
- Perfect for demonstrations and UI review
- For full functionality, see full deployment guide

TESTING TIPS:
- Field interface: Test on mobile device
- Manager interface: Test on desktop browser
- Try "Add to Home Screen" on mobile for PWA experience
